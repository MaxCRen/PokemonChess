representation of a battle

val get_player : t -> Pokemon.t

get_player battle returns the user's pokemon in the battle battle

val get_opponent : t -> Pokemon.t

get_opponent battle returns the opponent's pokemon in the battle battle

val make_battle : Pokemon.t -> Pokemon.t -> t

make_battle poke1 poke2 returns a new instance of a battle between pokemon poke1 and pokemon poke2

val get_move_from_str : Moves.t list -> string -> Moves.t

get_move_from_str moves move returns the move in moves corresponding to the string move

val can_move : Moves.t -> bool

can_move move returns true if the pp for move move > 0

val can_use_move : t -> string -> bool

can_use_move battle move is true if the current pokemon can use the move move otherwise it is false

val hit : Pokemon.t -> Moves.t -> float

hit poke move determines whether the pokemon poke hits or misses with the move move. It is 0 if it misses, and 1 if it hits

val calc_damage : Moves.t -> t -> float

calc_damge move battle calculates the amount of damage move does to the opponent pokemon in the battle. Take into account whether or not the move misses Calculation: (20* Move Power * (Pokemon Attack/Opponent Defense)/50)

val calc_effective : Moves.t -> Pokemon.t -> float -> float

calc_effective move poke dam calculates the effectiveness of move on pokemon poke and applies the necessary mutiliplier to the damage dam

val deal_damage : int -> Pokemon.t -> unit

deal_damage dam poke Deals dam to pokemon poke

val use_move : t -> Moves.t -> unit

use_move battle move gives us a new battle after using move
